Every answer flips to reveal the exact query behind it plus provenance — the judges' "no black box" moment. 5a is the full back of the 1a Divergence answer: the ClickHouse SQL, the source tables, rows scanned, query time and per-feed freshness. 5b shows the flip itself — the ⟲ view SQL chip on the front turns the card over (and back).

5aCard-back — the exact ClickHouse query + provenance
⟲ QUERY · is htmx hype or real?flip back ↩ · F
-- talk (HN) vs code (GH), z-scored on a shared hourly bucket
WITH talk AS (
  SELECT toStartOfHour(time) AS h, count() AS mentions
  FROM hackernews
  WHERE hasToken(lower(title), 'htmx') OR hasToken(lower(text), 'htmx')
    AND time > now() - INTERVAL 14 DAY
  GROUP BY h
),
code AS (
  SELECT toStartOfHour(created_at) AS h, count() AS events
  FROM github_events
  WHERE repo_name ILIKE '%htmx%'
    AND event_type IN ('PushEvent','PullRequestEvent','WatchEvent')
    AND created_at > now() - INTERVAL 14 DAY
  GROUP BY h
)
SELECT h,
       (mentions - avg_m) / std_m AS talk_z,
       (events   - avg_c) / std_c AS code_z
FROM talk FULL JOIN code USING (h)
ORDER BY h
