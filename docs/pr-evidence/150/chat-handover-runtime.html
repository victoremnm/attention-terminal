Chat handover runtime repair

PR 150 preview evidence for the failure reported from the Vercel deployment.

Observed failure

Before the fix, POST /api/chat returned 500 and the UI showed chat.handover endpoint returned 500.

Error [TriggerApiError]: No matching branch env
status: 401
branch=fix/146-chat-markdown

The Vercel runtime log identified the cause: the Trigger SDK inherited the Vercel preview branch, but that Trigger project had no matching preview branch environment.

Repair

attentionTriggerApiClient explicitly uses TRIGGER_PREVIEW_BRANCH when configured; otherwise it passes an empty preview branch so the SDK uses the default development environment instead of inferring VERCEL_GIT_COMMIT_REF.

The same client is now used by session creation, public-token minting, and the head-start route.
